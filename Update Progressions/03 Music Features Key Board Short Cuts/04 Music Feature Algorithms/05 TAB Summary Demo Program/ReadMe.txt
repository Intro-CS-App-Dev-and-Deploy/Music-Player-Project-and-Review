Music Features Reorganzied

- Shuffle Playlist Boolean to lock code only active during autoplay
- Mute, FF, FR, Pause: anytime

  - Mute will stop at end of Song, cue current song
- Quit will reset all variable and Booleans
- Next and Previous interrupt autoplay
- Same as STOP
- Play next or shuffle depends on Booleans Activated

Able to do a text file to restart variable
